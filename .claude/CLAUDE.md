# Claude Cowork — Sesión Formativa IA

Este directorio sirve como entorno y proyecto de demostración (Vault de Obsidian) para la **"Sesión formativa IA para equipos docentes"**. Contiene todos los recursos, guiones y ejemplos necesarios para impartir la clase teórico-práctica de 90 minutos.

## Estructura y Archivos Clave

- `Programa de la sesión.md`: El guion detallado y el timing completo de los bloques de la clase. 
- `Sesión Formativa IA.canvas`: Contiene el soporte visual completo y oficial para la clase. Es el tablero principal que se debe consultar/presentar durante la sesión.
- `README.md`: Documentación principal orientada a las *skills* educativas disponibles en este entorno (generación de temarios, corrección, rúbricas, etc.).
- `cata/`: Es **únicamente un proyecto de ejemplo** (sobre cuadernos de formación de Vander Coffee). Se utiliza internamente para hacer demostraciones sobre cómo la IA interactúa con repositorios y modifica archivos, pero no es el núcleo de la sesión educativa.
- `skills-preinstalados.canvas`: Tablero visual adicional sobre las herramientas y skills de la clase.

## Directrices para el Asistente (Claude)

1. **El Enfoque es Markdown:** El hilo conductor de la formación y de este repositorio es `Obsidian (.md) → Cowork + Skills → Claude Code → GitHub → Vercel`. Todo fluye a través de texto plano y `.md`.
2. **Perfil del Usuario:** El público objetivo son equipos docentes con un perfil no técnico. El enfoque está en cómo la IA amplifica su capacidad creadora y reduce el tiempo de adaptación al cambio tecnológico — permitiendo explorar nuevos formatos con más libertad y menos fricción.
3. **Manejo del Canvas:** `Sesión Formativa IA.canvas` tiene todos los "visuals" completos. Al ser un archivo de Obsidian Canvas (JSON subyacente), no lo modifiques directamente a menos que sea estrictamente necesario o se solicite expresamente.
4. **El Directorio `cata/`:** No confundas el proyecto real (la clase de IA y la docencia) con el contenido de `cata/`. Ese directorio es una simple sandbox/ejemplo técnico ajeno al temario de educación.
